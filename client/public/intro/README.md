# Intro Splash Assets

Place the following files here:

- `intro-video.webm` — WebM video (preferred format, smaller size)
- `intro-video.mp4` — MP4 fallback video
- `intro-audio.mp3` — Audio track for the intro
- `intro-poster.webp` — Poster/fallback image for the video

## Specifications

- **Video duration:** 4–8 seconds recommended
- **Resolution:** 1920×1080 or higher
- **Style:** Calm, premium, black-and-white logo reveal with light/haze effects
- **Audio:** Subtle ambient sound or gentle chime, ~5 seconds

The component will gracefully handle missing files with a CSS-only fallback animation.
