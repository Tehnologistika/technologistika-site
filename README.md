# ТехноЛогистика — сайт

Сайт компании **ТехноЛогистика** — премиальные логистические услуги (автовозы, доставка, перевозки) с корнями в Кабардино-Балкарии (Эльбрус).

Это одностраничное клиентское приложение (SPA) на React, которое в production-режиме раздаётся лёгким Express-сервером.

## Стек

- **Frontend:** React 19 + TypeScript, сборка через Vite 7
- **UI:** Tailwind CSS 4, Radix UI, Framer Motion, Wouter (роутинг)
- **Backend:** Express 4 (раздача статики и SPA-фолбэк), сборка через esbuild
- **Менеджер пакетов:** pnpm 10 (через Corepack)
- **Node.js:** версия 22 LTS

## Структура проекта

```
client/        исходники фронтенда (React, точка входа client/index.html)
server/        Express-сервер (server/index.ts)
shared/        код, общий для клиента и сервера
dist/          результат сборки
  ├── public/  собранный фронтенд (статика)
  └── index.js собранный сервер
```

## Требования

- Node.js **22 LTS**
- pnpm **10** (рекомендуется включить через Corepack: `corepack enable`)

## Установка зависимостей

```bash
pnpm install
```

## Запуск в режиме разработки

```bash
pnpm dev
```

Vite поднимет dev-сервер с hot-reload и выведет локальный адрес (по умолчанию `http://localhost:5173`). Флаг `--host` уже включён в скрипте, поэтому сайт доступен и по адресу в локальной сети.

## Проверка типов

```bash
pnpm check
```

Запускает `tsc --noEmit` — проверяет типы без генерации файлов. Используйте перед коммитом и в CI.

## Production-сборка

```bash
pnpm build
```

Команда выполняет два шага:

1. `vite build` — собирает фронтенд в `dist/public`.
2. `esbuild server/index.ts ...` — бандлит Express-сервер в `dist/index.js`.

## Запуск production

```bash
pnpm start
```

Поднимает Express-сервер (`NODE_ENV=production node dist/index.js`), который раздаёт статику из `dist/public` и отдаёт `index.html` на все маршруты (клиентский роутинг).

Порт задаётся переменной окружения `PORT` (по умолчанию `3000`).

## Переменные окружения

Скопируйте пример и при необходимости отредактируйте:

```bash
cp .env.example .env
```

| Переменная     | Назначение                                   | Пример                         |
| -------------- | -------------------------------------------- | ------------------------------ |
| `NODE_ENV`     | Режим работы сервера                         | `production`                   |
| `PORT`         | Порт Express-сервера                         | `3000`                         |
| `SITE_DOMAIN`  | Основной домен сайта                         | `технологист.рф`               |
| `AYUB_API_URL` | Адрес внешнего API (поддомен `ayub`)         | `https://ayub.технологист.рф`  |
| `TELEGRAM_BOT_TOKEN`     | Токен Telegram-бота для отправки заявок с формы | `123456:ABC-DEF...`        |
| `TELEGRAM_LEADS_CHAT_ID` | ID чата/группы, куда приходят заявки            | `-1001234567890`           |
